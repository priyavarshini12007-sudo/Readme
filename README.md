Monitoring System for Classroom Session in Skill Training Programme

Problem Statement
Manual attendance and classroom session monitoring are time-consuming and difficult to maintain.

Objective
- Monitor classroom sessions
- Maintain attendance records
- Calculate attendance statistics

Features
- Add Student
- Search Student
- Mark Attendance
- Session Management
- Dashboard
- Attendance Percentage
- Date and Time Display

Technologies Used
- HTML
- CSS
- JavaScript

Advantages
- Saves time
- Easy attendance management
- Digital record maintenance
- User-friendly interface

Future Scope
- Login authentication
- Export attendance reports
- Database integration
